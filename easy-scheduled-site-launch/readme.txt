Display a customizable coming soon page with countdown, logo, and templates until your scheduled launch date.

== Description ==
**Easy Scheduled Site Launch** allows you to schedule your WordPress website to go live automatically on a specific date and time.  
Until then, visitors will see a sleek and customizable *Coming Soon* page while administrators can continue editing the site.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/easy-scheduled-site-launch/` or install it via the WordPress Plugin Installer.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Go to **Settings ‚Üí Scheduled Launch** to configure your launch date, message, logo, and design preferences.

== Frequently Asked Questions ==

= Will this block administrators? =  
No. Logged-in admins with the `manage_options` capability always see the live site.

= Can I disable the countdown timer? =  
Yes. In settings, simply select ‚ÄúNo‚Äù under **Enable Countdown Timer**.

= What happens after the launch date passes? =  
The coming soon mode automatically turns off, and your website becomes fully visible to everyone.

= Does this work with WooCommerce? =  
Yes, it is fully compatible with WooCommerce stores.
